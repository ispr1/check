# User Personas

## 1. Candidate: "Rohan" (The Applicant)
*   **Profile:** Software Engineer, 4 years experience. Switching jobs for a hike.
*   **Goal:** Get verified quickly without needing to courier physical documents or visit an office.
*   **Frustrations:** Uploading the same data multiple times; waiting weeks for offer letter release due to BGV.
*   **Interaction:** Receives "Magic Link" -> Uploads Selfie, IDs, Docs -> Done in 5 mins.

## 2. HR Operations: "Priya" (The Reviewer)
*   **Profile:** Senior HR Associate, handles 50+ onboardings/week.
*   **Goal:** Filter out bad actors efficiently; approve genuine candidates instantly.
*   **Frustrations:** Reading complex vendor reports; vague "Red/Amber/Green" signals; legal liability of bad hires.
*   **Interaction:** Dashboard View -> Sees "Trust Score 45 (Flagged)" -> Drills down -> Sees "Face Mismatch" -> Rejects.

## 3. Risk Officer: "Anil" (The Auditor)
*   **Profile:** VP of Compliance.
*   **Goal:** Ensure ISO27001/SOC2 compliance; ensure audit trails exist for every hiring decision.
*   **Frustrations:** Missing data; unlogged overrides ("Why did we hire him if he failed ID check?").
*   **Interaction:** API Audit Logs -> Checks "Who overrode this score?" -> Exports Compliance Report.
